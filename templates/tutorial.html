使用教程

1. 创建虚拟机构

在虚拟机构卡片中，您可以轻松创建新的CA证书：

  • 点击"创建"按钮；
  • 填写机构名称（可留空）和密码（可不设）；
  • 点击确认生成CA证书；
  • 下载虚拟机构的CA证书。

2. 证书申请

在证书列表卡片中，您可以随时申请新的SSL证书：

  • 点击"新增"按钮；
  • 填写IP地址或域名信息（支持多IP和多域名）；
  • 填写证书的基本信息（CA证书如果设了密码要填写密码）；
  • 点击确认生成证书。

3. 证书验证

在证书验证页面中，您可以：

  • 先安装CA证书到本地设备；
  • 填写本地设备IP地址，如：192.168.0.2；
  • 选择证书列表里的证书或上传证书文件（证书里的IP地址需包含有本地设备IP）；
  • 点击"验证"，稍等一会自动跳转验证页面；
  • 如提示不安全链接，关闭浏览器，重新验证。

3. 反向代理配置

在反向代理页面中，您可以：

  • 配置HTTP转HTTPS反向代理服务，如：http://192.168.0.2:5244反代成https://192.168.0.2:5245；
  • 其中http://192.168.0.2:5244为原本地址，https://192.168.0.2:5245为反代后地址；
  • 点击"新增"，启用SSL/TLS加密通信；
  • 开启或关闭代理服务。

4. 用户设置

在设置页面中，您可以方便地修改用户名和密码：

  • 填入新的用户名；
  • 填入旧的密码和想要创建的新密码；
  • 点击"保存"修改。

各系统安装CA证书的方法

1. Windows系统

在Windows系统中安装CA证书：

  • 双击下载的CA证书文件；
  • 点击"安装证书"；
  • 选择"本地计算机"，点击"下一步"；
  • 选择"将所有的证书都放入下列存储"，点击"浏览"；
  • 选择"受信任的根证书颁发机构"，点击"确定"；
  • 点击"下一步"，然后点击"完成"。

2. Linux系统

在Linux系统中安装CA证书：

  • 将CA证书复制到/usr/local/share/ca-certificates/目录；
  • 执行命令：sudo update-ca-certificates；
  • 重启浏览器即可生效。

3. Android系统

在Android系统中安装CA证书：

  • 下载CA证书到手机；
  • 打开设置，搜索"证书"或"安全"；
  • 选择"安装证书"或"从存储设备安装"；
  • 找到并选择下载的CA证书文件；
  • 按照提示完成安装。

4. iOS系统

在iOS系统中安装CA证书：

  • 用Safari浏览器下载CA证书；
  • 打开设置，点击"已下载描述文件"；
  • 点击"安装"按钮；
  • 输入设备密码；
  • 前往设置-通用-关于本机-证书信任设置；
  • 找到新安装的CA证书并启用信任。
